# Business Skills

Repositorio publico de skills para empresarios, operadores e desenvolvedores que usam OpenClaw/Claw para automacoes de negocio.

## Objetivo

Reunir skills reutilizaveis para operar SaaS, marketing, vendas, customer success, suporte, conteudo, BI, integracoes e rotinas executivas com agentes.

As skills deste repositorio nasceram em operacao real e devem ser tratadas como playbooks: adapte nomes, canais, ferramentas, politicas e limites antes de usar em outro negocio.

## Estrutura

- `skills/`: skills em formato OpenClaw, cada uma com seu proprio `SKILL.md`.
- `references/`: materiais auxiliares dentro de algumas skills.

## Como Usar

1. Escolha uma skill em `skills/<nome>/SKILL.md`.
2. Leia os objetivos, guardrails e criterios de uso.
3. Copie para o workspace de skills do seu agente ou adapte ao seu padrao.
4. Substitua referencias especificas de marca, ferramenta, ERP, canal e responsaveis.
5. Teste em ambiente controlado antes de automatizar acoes externas.

## Cuidados

- Nao publique credenciais, tokens, cookies, arquivos de secrets, dados pessoais, bases de clientes ou prints internos.
- Skills comerciais e juridicas nao substituem decisores, contador, advogado, DPO ou responsavel tecnico.
- Acoes externas como posts, DMs, WhatsApp, emails, disparos, deploys e alteracoes financeiras devem exigir aprovacao humana quando houver risco.
- Para repositorios publicos, prefira exemplos genericos e caminhos configuraveis.

## Contribuicao

Novas skills devem incluir:

- nome claro;
- descricao curta;
- quando usar;
- quando nao usar;
- procedimento operacional;
- guardrails de seguranca;
- criterio de pronto ou evidencia.
