# Conecta Tutoriais Demo Captura

Use para criar tutoriais, prints, gravacoes de tela, roteiros de demo, videos curtos e materiais de onboarding/marketing do Conecta Clube.

## Mandato

Transformar o produto em explicacoes visuais claras para lojistas, equipe comercial, suporte e marketing.

## Seguranca

- Usar apenas contas e dados autorizados.
- Nao expor senhas, tokens, dados pessoais reais ou informacoes sensiveis.
- Preferir ambiente/teste ou loja demo.
- Se aparecer dado real, borrar/ocultar antes de publicar.
- Nao executar acoes destrutivas durante captura.

## Fluxo De Captura

1. Definir objetivo do material.
2. Definir publico: lojista, vendedor, suporte, cliente final ou representante.
3. Preparar roteiro curto.
4. Entrar no ambiente autorizado.
5. Limpar notificacoes/dados sensiveis.
6. Capturar telas ou gravar fluxo.
7. Revisar se existe informacao sensivel.
8. Gerar legenda, titulo e CTA.
9. Salvar fonte e versao.

## Tutoriais Prioritarios

- Primeiro acesso.
- Configurar programa de pontos.
- Cadastrar cliente.
- Dar pontos.
- Criar recompensa.
- Enviar campanha WhatsApp.
- Compartilhar QR/link.
- Ver relatorios.
- Usar CRM.
- Configurar catalogo.

## Roteiro Padrao De Video

- Dor em uma frase.
- O que vamos fazer.
- Passo a passo em tela.
- Resultado esperado.
- Erro comum.
- Proxima acao.

## Artefatos

- Roteiro de video.
- Checklist de captura.
- Prints anotados.
- Script de narracao.
- Legendas curtas.
- Thumbnail/capa.
- Lista de tutoriais pendentes.

## Regras

- Material para lojista deve ser curto, direto e sem jargao tecnico.
- Tutorial bom termina com uma acao real no produto.
- Marketing pode usar tela, mas precisa vender resultado: recompra, rotina e clientes voltando.
- Se a UI mudar, revisar tutorial afetado.
