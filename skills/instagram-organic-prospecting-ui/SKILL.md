# Instagram Prospeccao Organica UI

## Regra Obrigatória Para Destaques

Nunca criar ou atualizar um destaque usando stories antigos apenas porque estão disponíveis no arquivo. Destaque é uma vitrine permanente e precisa ter contexto, sequência e intenção de conversão.

Antes de criar um destaque:

1. Definir o objetivo do destaque.
2. Criar os criativos específicos para esse objetivo.
3. Revisar ortografia, acentuação e CTA.
4. Publicar esses criativos primeiro nos stories.
5. Só depois adicionar esses stories ao destaque.
6. Conferir o destaque público após salvar.

## Proibido

- Usar story antigo fora de contexto.
- Misturar criativos de campanhas diferentes no mesmo destaque.
- Criar destaque apenas com materiais genéricos ou datados.
- Criar destaque sem ter publicado antes os criativos como story.
- Improvisar capa ou texto se a sequência não estiver clara.

## Sequência Recomendada Para `Como funciona`

Story 1: `Fidelização simples para comércio local`

- Cliente compra e se cadastra.
- Ganha pontos ou benefícios.
- Recebe campanhas para voltar.
- CTA: recorrência sem depender só de desconto.

Story 2: `Sua base deixa de ficar escondida`

- Quem comprou.
- Quem voltou.
- Quem sumiu e precisa de ação.
- CTA: decisão melhor começa com cliente organizado.

Story 3: `Descubra onde a recompra vaza`

- Diagnóstico gratuito.
- Mapa rápido da base.
- Plano para reativar clientes.
- CTA: peça pelo link da bio ou chame no direct.

## Limitação Do Instagram Web

Se o Instagram Web não mostrar opção de publicar story:

1. Não publicar como feed por engano.
2. Não criar destaque com stories antigos.
3. Gerar os criativos locais corretamente.
4. Deixar registrado o caminho dos arquivos.
5. Aguardar publicação via app/celular ou outra interface que permita story.
6. Depois da publicação, criar o destaque com esses stories recém-publicados.

## Registro

Sempre registrar:

- Nome do destaque.
- Criativos usados.
- Se foram publicados como story.
- Se entraram no destaque.
- Limitações da interface.
- Próximo passo pendente.
